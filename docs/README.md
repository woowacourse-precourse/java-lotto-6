# 미션: 로또

## 구현 기능 목록

### 입출력
- 입력
    - [ ] 로또 구입 금액 입력
    - [ ] 당첨번호 입력
    - [ ] 보너스 번호 입력

- 출력
    - [ ] 입력 요청 메시지 출력
    - [ ] 발행 로또숫자 및 번호 출력(로또 번호는 오름차순)
    - [ ] 당첨 내역(통계),수익률(소수 둘째자리에서 반올림) 출력
    - [ ] 예외 메시지 출력([ERROR]로 시작)

### 로또 회사
- [X] 로또 발행(n개)
- [X] 당첨 내역 확인
  - [X] 각 로또 당첨 내역 확인
  - [X] 전체 당첨 내역 확인

### 캐셔
- [X] 발행 로또 개수 확인
- [X] 총 당첨금 반환
- [ ] 수익률 계산
