# 3주 차 프리코스 미션 - 로또

📃해당 프로젝트는 간단한 로또 게임을 구현한 프로젝트입니다.
<br>

# 📬입력 예시

## **숫자가 아닌 입력 시 공통 예외**

- [wrong]`~ 입력해 주세요.`

```text
일,이,삼,사,오,육 or 문자열
[ERROR] 숫자만 입력 가능합니다.
```

## **로또 구입 금액을 입력받는다.**

- 구입 금액은 `1,000원 단위`로 입력받는다.
- 최대 구입 금액은 `100,000`원이다.
- 최소 구입 금액은 `1,000`원이다.

### **구입 금액 입력 예시**

- [right]`구입금액을 입력해 주세요.`

```text
8000
```

```text
10000
```

- [wrong]`구입금액을 입력해 주세요.`

```text
500
[ERROR] 로또 구입 금액은 최소 1,000원입니다. 
```

```text
110000
[ERROR] 최대 구입 가능 금액은 100,000원입니다. 
```

```text
5555
[ERROR] 1,000원 단위로 입력해 주세요.
```

## **당첨 번호를 입력받는다.**

- `6개의 숫자`가 입력되어야 한다.
- `쉼표(,) 기준`으로 구분한다.
- `중복되지 않아야` 한다.
- `1부터 45 사이의 숫자`여야 한다.

### **당첨 번호 입력 예시**

- [right]`당첨 번호를 입력해 주세요.`

```text
1,2,3,4,5,6
```

```text
45,43,42,41,40,39
```

- [wrong]`당첨 번호를 입력해 주세요.`

```text
1,2,3,4,5 or 1,2,3,4,5,6,7
[ERROR] 로또 번호는 6개여야 합니다.
```

```text
1 2 3 4 5 6 or 123456
[ERROR] 쉼표(,)로 구분해 주세요.
```

```text
1,1,2,3,4,5
[ERROR] 로또 번호는 중복될 수 없습니다.
```

```text
1,2,3,4,5,46 or 0,1,2,3,4,5
[ERROR] 로또 번호는 1부터 45 사이의 숫자여야 합니다.
```

## **보너스 번호를 입력받는다.**

- `당첨 번호와 중복되지 않아야` 한다.
- `1부터 45 사이의 숫자`여야 한다.

### **보너스 번호 입력 예시**

`ex) 당첨 번호 : [1, 2, 3, 4, 5, 6]`

- [right]`보너스 번호를 입력해 주세요.`

```text
7
```

- [wrong]`보너스 번호를 입력해 주세요.`

```text
6
[ERROR] 보너스 번호는 당첨 번호와 중복될 수 없습니다.
```

```text
50
[ERROR] 로또 번호는 1부터 45 사이의 숫자여야 합니다.
```

# 기능 목록

## 🧱Constant

### 게임 관련 상수 (GameConfig)

```text
- ✅ 최대 구입 금액
- ✅ 로또 한 장의 가격
- ✅ 당첨 번호의 개수
- ✅ 로또 번호의 최소 값
- ✅ 로또 번호의 최대 값
- ✅ ZERO 값
```

### 로또 메시지 관련 상수 (LottoMessage)

```text
- ✅ 구입 금액을 입력하는 메시지
- ✅ 구매한 로또 수량을 출력하는 메시지
- ✅ 로또 번호를 출력하는 메시지
- ✅ 당첨 번호를 입력하는 메시지
- ✅ 보너스 번호을 입력하는 메시지
- ✅ 당첨 통계를 출력하기 전 통계 Title을 출력하는 메시지
- ✅ 당첨 통계를 출력하는 메시지
- ✅ 총 수익률을 출력하는 메시지
```

### 예외 관련 상수 (ExceptionMessage)

```text
- ✅ 숫자가 아닌 입력이 들어온 경우 예외 메시지
- ✅ 구입 금액이 1,000원 단위가 아닌 경우 예외 메시지
- ✅ 최대 구입 금액인 100,000원을 초과한 경우 예외 메시지
- ✅ 구입 금액이 1,000원 미만인 경우 예외 메시지
- ✅ 쉼표(,)로 구분되지 않은 경우 예외 메시지
- ✅ 1부터 45 사이의 숫자가 아닌 경우 예외 메시지
- ✅ 중복된 숫자가 입력된 경우 예외 메시지
- ✅ 6개의 숫자가 입력되지 않은 경우 예외 메시지
- ✅ 당첨 번호와 중복된 보너스 볼을 입력한 경우 예외 메시지
```

### 기호 관련 상수 (CommonSymbol)

```text
- ✅ "," 기호
```

## ⚙️model

### PurchaseAmount

```text
- ✅ 구입 금액을 저장한다.
- ✅ 구입 금액의 유효성을 검사한다.
- ✅ 구입 가능한 로또의 수량을 계산한다.
- ✅ 구입한 로또만큼 금액을 차감한다.
- ✅ 구입한 로또의 수량을 출력한다.
```

### Lotto

```text
- ✅ 로또 번호의 유효성을 검사한다.
- ✅ 로또 번호를 오름차순으로 정렬한다.
- ✅ 로또 번호를 출력한다.
- ✅ 당첨번호와 비교하여 등수를 반환한다
```

### Lotto's

```text
- ✅ 로또 번호를 생성한다.
- ✅ 금액에 해당하는 로또를 생성한다.
- ✅ 생성한 로또만큼 금액을 차감한다.
- ✅ 로또들의 번호를 출력한다.
- ✅ 가지고 있는 로또들의 등수들을 WinningResult로 반환한다.
```

### WinningNumber

```text
- ✅ 당첨 번호를 저장한다.
- ✅ 당첨 번호의 유효성을 검사한다.
- ✅ 당첨 번호를 반환한다.
```

### BonusNumber

```text
- ✅ 보너스 번호를 저장한다.
- ✅ 보너스 번호의 유효성을 검사한다.
- ✅ 보너스 번호를 반환한다.
```

### WinningLotto

```text
- ✅ 당첨 번호와 보너스 번호를 저장한다.
- ✅ 당첨 번호와 보너스 번호의 유효성을 검사한다.
- ✅ 당첨 번호와 보너스 번호를 반환한다.
```

### [enum] LottoRank

```text
- ✅ 등수를 나열한다.
- ✅ 등수에 해당하는 상금을 반환한다.
- ✅ 당첨 번호와 보너스 번호를 비교하여 등수를 반환한다.
```

### WinningResult

```text
- ✅ Map 형태로 등수별 당첨 횟수를 추가한다.
- ✅ 당첨 통계를 출력한다.
- ✅ 총 수익률을 계산한다.
- ✅ 총 수익률을 출력한다.
```

### ProfitRate

```text
- ✅ 총 수익률을 저장한다.
- ✅ 수익률을 계산한다.
- ✅ 총 수익률을 반환한다.
```
