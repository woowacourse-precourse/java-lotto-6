# 로또 게임 
> 작성자 : 배영석

### 기능 목록
- [x] 금액 입력
- [ ] 로또 구입
- [ ] 당첨 번호 및 보너스 번호 입력
- [ ] 로또 발행
- [ ] 당첨 번호 추첨
- [ ] 일치하는 번호 숫자와 보너스 숫자 일치 여부 판별
- [ ] 당첨 내역 출력
- [ ] 수익률 출력
- [ ] 예외 문구 출력

### 테스트 목록
- [x] 금액이 1000원 단위가 아니면 예외처리한다.
- [ ] 당첨번호를 입력받는다.
- [ ] 로또 번호를 오름차순으로 출력한다.
- [ ] 당첨 내역을 출력한다.
- [ ] 수익률을 출력한다.
- [ ] 입력값이 1부터 45인지 확인한다.
