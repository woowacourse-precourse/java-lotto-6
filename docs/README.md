# 로또 (조형우)

---

## 기능 목록

### 수익률
    - 소수점 둘째 자리에서 반올림한다.

### 당첨 숫자 비교
    - 입력된 당첨 숫자를 토대로 몇 개가 일치하는지 확인한다.

### 로또 추첨
    - 6개의 1 ~ 45 사이의 숫자를 랜덤으로 생성한다.
    - 생성된 숫자를 오름차순으로 다시 정렬한다.
    
### 당첨 통계
    - 구입한 로또에 대한 당첨 결과를 수익률과 함께 출력한다.

### 예외 사항
    1. 구매 금액은 1000원 단위여야 한다.
    2. 당첨 번호는 중복되면 안된다.
    3. 당첨 번호는 1 ~ 45 이다.
    4. 입력은 숫자만 해야 한다.
