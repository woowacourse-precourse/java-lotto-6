# 로또
## 개요
- 구매금액을 입력하면 금액에 맞는 게임 수 만큼 자동 로또 번호를 생성하고,   
해당 회차의 당첨 번호를 입력받아 구매한 게임의 당첨 통계를 출력하는 프로그램을 작성한다.   

## 기능목록   
1. 로또 구입금액 입력 받기
    - 1000원 단위로 입력, 아니면 예외처리
2. 발행한 로또 수량 출력
3. 발행한 로또 번호 출력
    - 로또 번호 오름차순 정렬
4. 당첨 번호 입력 받기
    - 쉼표로 구분
5. 보너스 번호 입력 받기
6. 당첨 결과 출력하기
    - 당첨 내역 출력
    - 수익률 출력
        + 소수점 둘째자리에서 반올림
