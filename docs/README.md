# 구현할 기능 목록

## 입력

---

### 입력 공통 사항
- [x] exception 발생 시 `[ERROR] error message` 출력 후 그 부분부터 다시 입력 
---
- [x] 로또 구입 금액 입력
  - 1000원 단위
  - Error
    - [x] 1000으로 나눴을 때 나머지가 남을 경우
    - [x] 문자인 경우 
    - [x] 음수나 0의 경우

-[x] 당첨 번호 입력
  - 쉼표를 기준으로 구분
  - Error
    - [x] split 후 개수가 6개가 아닐 때
    - [x] split 후 숫자가 중복될 때
    - [x] split 후 문자일 경우
    - [x] split 후 숫자가 1~45가 아닐 경우
  
- [x] 보너스 번호 입력
  - 1~45 입력
  - Error
    - [x] 숫자가 아닌 문자가 있을 경우
    - [x] 숫자가 1~45 가 아닌 경우
    - [x] 숫자가 당첨 번호랑 중복인 경우
