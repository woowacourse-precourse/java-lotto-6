## 로또 미션

## 기능 구현 목록

- [ ] 로또 구입 금액을 입력 받음.
  - [ ] 예외처리 : 숫자가 아닌 경우 예외처리.
  - [ ] 예외처리 : null, blank인 경우 예외처리.
  - [ ] 예외처리 : 1000이상의 양수가 아닌 경우 예외처리.
  - [ ] 예외처리 : 1000으로 나누어 떨어지지 않으면 예외처리.
- [ ] 입력 금액에 따라서 발행할 로또 개수 계산.
- [ ] 발행 개수에 따라서 로또 구매.
- [ ] 당첨 번호를 입력 받음.
- [ ] 보너스 번호를 입력 받음.
- [ ] 당첨 금액 계산
