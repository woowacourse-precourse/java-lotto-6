**lotto**

- [X] lotto ticket으로써 6가지 숫자를 생성자에서 받는다.
- [X] 6가지 숫자와 당첨번호, 보너스 당첨번호를 비교하여 몇개가 일치하는지 저장한다.

**random**

- [X] 중복되지 않는 랜덤숫자들을를 반환한다. lottoNumbers = generateLottoNumbers()
