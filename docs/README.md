# 미션 - 로또

---

## 구현 기능 목록

### 입력 기능

- 로또 구입 금액 입력 기능
- 당첨 번호 입력 기능
- 보너스 번호 입력 기능

### 계산 기능

- 입력한 금액 1,000원당 로또 1개를 발행하는 기능
- 발행한 로또와 입력한 로또 비교 기능

### 출력 기능

- 구입한 로또 출력 기능
    - 구입한 로또 개수 출력
    - 구입한 로또 번호 출력
- 당첨 내역 출력 기능
- 총 수익률 출력 기능

### 예외 검사 기능

#### [입력값 예외 검사]

- 구입 금액
    - 구입 금액이 1,000원 단위인지 검사
    - 구입 금액이 숫자인지 검사
- 당첨 번호
    - 당첨 번호가 쉼표로 구분되었는지 검사
    - 당첨 번호가 쉼표를 제외하고 입력값 길이가 6인지 검사
    - 당첨 번호가 숫자인지 검사
    - 당첨 번호가 1 이상, 45 이하인지 검사
- 보너스 번호
    - 보너스 번호의 길이가 1인지 검사
    - 보너스 번호가 숫자인지 검사
    - 보너스 번호가 1 이상, 45 이하인지 검사
    - 보너스 번호가 당첨 번호와 중복되었는지 검사
