## 기능 목록
- [x] 로또를 구입한다.
  - [x] 구입 금액은 숫자여야 한다.
  - [x] 1000원 단위인지 확인한다.


- [x] 당첨 번호를 추첨한다.
    - [x] 1부터 45 사이의 중복되지 않는 무작위 값 6개와 보너스 번호 1개를 입력받는다.
        - [x] 중복되지 않는지 확인한다.
        - [x] 보너스 번호 또한 당첨 번호에 포함되지 않는지(중복) 확인한다.

- [x] 당첨 여부를 확인한다.
    - [x] 일반 번호 일치 개수를 구한다.
    - [x] 보너스 번호 일치 여부를 구한다.
    - [x] 등수에 따라 상금을 지급한다.

- [x] 상금에 따른 수익률을 계산한다. (수익률 = 상금 / 구입 금액)
    - [x] 수익률은 소수점 둘째 자리에서 반올림한다.
