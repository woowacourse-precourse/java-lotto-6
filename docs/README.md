# 기능 목록

## 기능
- [ ] 구입 금액 입력
- [ ] 개수만큼 로또 생성
- [ ] 랜덤으로 만들어진 로또 보여줌
- [ ] 당첨 번호 입력
- [ ] 보너스 번호 입력
- [ ] 생성된 로또의 결과 계산
- [ ] 결과를 종합하여 당첨 통계 생성
- [ ] 당첨 통계를 통해 수익률 계산
- [ ] 입력을 검증
  - [ ] 구입 금액
    - [ ] 숫자로 이루어져야 함
    - [ ] 1000원 단위로 나누어져야 함
  - [ ] 로또 번호
    - [ ] 쉼표(,)로 구분된 6개의 수
    - [ ] 각 수는 1부터 45의 수
  - [ ] 보너스 번호
