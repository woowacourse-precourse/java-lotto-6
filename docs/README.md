# 테스트 케이스 예상

1. 금액 입력받기
   
   ```
   "3,000"
   "3400"
   "800"
   "8천원"
   ```

2. 당첨번호 6개 입력받기
   
   ```
   "3,4,5,6,7,8,9"
   "46,47,5,6,3,2"
   "4,,5,3,2,5"
   "4, ,5,3,2,5"
   "2,3,4"
   ```

3. 보너스 번호 입력받기
   
   ```
   "3,4,5,6,7,8"
   "47"
   ""
   " "
   ```
